Data Visualization Project 1

About this Site:

This site and the data visualizations on it are part of our Data Visualizations class. The goal of this project, and the class in general, is to familiarize ourself with data visualization best practices, and to learn how to better handle and display data. Data visualization can be used to provide evidence for a message, and even simple graphics can be highly persuasive. For this reason, we should be cautious that are visualizations are unbiased and properly utilized.

To use this site, simply click on any of the links below to view the visualizations and learn more about the purpose and functionality of differnt chart types.

The data used in following visualizations was either created by me or comes from external sources, in which case links to the sources are given.
